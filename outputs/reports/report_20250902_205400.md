# Review Report

### 외피_형별성능내역 (Envelope Performance by Type)

*   **Item:** W1
*   **Field:** 열교방지구조 (Thermal Bridge Prevention)
*   **Issue:** The `열교방지구조` is not checked in the model.
*   **Analysis:** According to the `형별성능관계내역 도면`, the material composition for wall type W1 is `실외표면저항 → 비드법보온판 1종1호 → 철근콘크리트 → ...` [1]. Since the insulation is located on the exterior side of the structural concrete, the `열교방지구조` option should be applied.
*   **Correction:** For the `W1` envelope type, check the box for `열교방지구조`.

*   **Item:** F2
*   **Field:** 열교방지구조 (Thermal Bridge Prevention)
*   **Issue:** The `열교방지구조` is not checked in the model.
*   **Analysis:** According to the `형별성능관계내역 도면`, the material composition for floor type F2 is `실외표면저항 → 버림콘크리트 → 비드법보온판 1종1호 → 콘크리트 슬라브 → ...` [1]. Since the insulation is located on the exterior side of the main concrete slab, the `열교방지구조` option should be applied.
*   **Correction:** For the `F2` envelope type, check the box for `열교방지구조`.

*   **Item:** F2
*   **Field:** 바닥난방여부 (Floor Heating Status)
*   **Issue:** The `바닥난방여부` is not checked in the model.
*   **Analysis:** The mechanical drawings indicate that a floor heating system is installed in '숙소-1' and '숙소-2' on the 2nd floor [2, 3]. The floor type `F2` corresponds to the 2nd-floor slab where these rooms are located.
*   **Correction:** For the `F2` envelope type, check the box for `바닥난방여부`.

### 외피 (Envelope)

*   **Item:** W2
*   **Field:** 방위 (Orientation)
*   **Issue:** All wall entries of type `W2` are incorrectly set to `일사없음` (No Solar Radiation).
*   **Analysis:** The model lists a total area of 75.33 m² for `W2` walls, all with the orientation `일사없음`. However, the `단열면적산출표` shows that these walls are directly exposed to the outside with specific orientations and areas [4].
*   **Correction:** Delete the existing `외피` entries for `W2`. Create new entries with the correct orientation and area as follows:
    *   `형별성능내역`: W2, `방위`: 남, `면적(m²)`: 38.61
    *   `형별성능내역`: W2, `방위`: 동, `면적(m²)`: 12.96
    *   `형별성능내역`: W2, `방위`: 서, `면적(m²)`: 23.76

*   **Item:** 바닥(최하층)
*   **Field:** 면적(m²) (Area)
*   **Issue:** The area for the lowest floor is incorrect.
*   **Analysis:** The model input for the `F1` floor area is 224.91 m². The `단열면적산출표` specifies the area as 216 m² [4].
*   **Correction:** For the `외피` entry corresponding to `바닥(최하층)` (F1), change the `면적(m²)` value from 224.91 to 216.

*   **Item:** 바닥(피트하부)
*   **Field:** 면적(m²) (Area)
*   **Issue:** The area for the floor above the pit is incorrect.
*   **Analysis:** The model input for the `F2` floor area is 26.73 m². The `단열면적산출표` specifies the area as 35.64 m² [4].
*   **Correction:** For the `외피` entry corresponding to `바닥(피트하부)` (F2), change the `면적(m²)` value from 26.73 to 35.64.
