# Review Report

### 기계설비부문_난방기기 (Mechanical Equipment_Heating Equipment)

*   **Item: YH-1**
    *   **Issue**: This equipment is incorrectly included in the evaluation.
    *   **Reason**: The equipment schedule [2] indicates that the purpose of `YH-1` is "화장실 동파방지" (toilet freeze prevention). According to the modeling manual, equipment used for freeze prevention is not subject to evaluation and should be excluded from the model [1].
    *   **Correction**: Please delete the `YH-1` entry from the heating equipment list.

*   **Item: EHPS - 14**
    *   **Field**: `성적계수(COP)` (Coefficient of Performance)
    *   **Issue**: The input value is incorrect.
    *   **Reason**: The model input for the COP is `2`. However, the COP must be calculated based on the equipment's performance data. According to the equipment schedule [3], the `정격난방용량` (Rated Heating Capacity) is 36,980 kcal/h (which is 43.0 kW) and the `정격소비전력` (Rated Power Consumption) is 13.0 kW. The correct COP is calculated as Capacity / Power Consumption.
    *   **Correction**: The `성적계수(COP)` should be changed from `2` to `3.31` (43.0 kW / 13.0 kW).

*   **Item: B-1**
    *   **Field**: `기기용량(kW)` (Equipment Capacity)
    *   **Issue**: The input value is incorrect.
    *   **Reason**: The model input for capacity is `16.83` kW. The equipment schedule [2] lists the `정격난방용량` (Rated Heating Capacity) as 14,500 kcal/h. This value must be converted to kW for input (1 kW = 860 kcal/h).
    *   **Correction**: The `기기용량(kW)` should be changed from `16.83` to `16.86` (14,500 kcal/h / 860).

*   **Item: B-1**
    *   **Field**: `기기효율(%)` (Equipment Efficiency)
    *   **Issue**: The input value is incorrect.
    *   **Reason**: The model input for efficiency is `70`%. The manual requires calculating the efficiency based on performance data [1]. The equipment schedule [2] specifies a `정격난방용량` (Rated Heating Capacity) of 14,500 kcal/h, a `연료소비량` (Fuel Consumption) of 1.97 L/h, and a fuel `발열량` (Calorific Value) of 8,300 kcal/L. The efficiency is calculated as Rated Heating Capacity / (Fuel Consumption × Calorific Value).
    *   **Correction**: The `기기효율(%)` should be changed from `70` to `88.7` (14,500 kcal/h / (1.97 L/h × 8,300 kcal/L) × 100).

### 기계설비부문_냉방기기 (Mechanical Equipment_Cooling Equipment)

*   **Item: EHPS - 14**
    *   **Issue**: The equipment is entered twice.
    *   **Reason**: The model lists two identical entries for `EHPS - 14`. However, the equipment schedule [3] confirms there is only one unit (`대수: 1`).
    *   **Correction**: Please delete one of the two identical `EHPS - 14` entries.

*   **Item: EHPS - 14**
    *   **Field**: `열성능비(COP)` (Coefficient of Performance)
    *   **Issue**: The input value is incorrect.
    *   **Reason**: The model input for the COP is `2`. The COP must be calculated based on the equipment's performance data. According to the equipment schedule [3], the `정격냉방용량` (Rated Cooling Capacity) is 34,916 kcal/h (which is 40.6 kW) and the `정격소비전력` (Rated Power Consumption) is 13.5 kW. The correct COP is calculated as Capacity / Power Consumption.
    *   **Correction**: The `열성능비(COP)` for the remaining `EHPS - 14` entry should be changed from `2` to `3.01` (40.6 kW / 13.5 kW).
