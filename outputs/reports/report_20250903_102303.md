# Review Report

## 기계설비부문_공조기기 (Mechanical Equipment_AHU)

*   **Item: EF-1, Field: 배기풍량(CMH)**
    *   The value entered in the model is 3.5.
    *   The equipment schedule [2] indicates the airflow for `EF-1` is 3.5 CMM. The model requires the unit to be CMH. Based on the unit conversion standard [1], 1 CMM = 60 CMH.
    *   Therefore, the `배기풍량(CMH)` must be corrected to **210**.

*   **Item: EF-2, Field: 배기풍량(CMH)**
    *   The value entered in the model is 9.
    *   The equipment schedule [2] indicates the airflow for `EF-2` is 9 CMM. The model requires the unit to be CMH. Based on the unit conversion standard [1], 1 CMM = 60 CMH.
    *   Therefore, the `배기풍량(CMH)` must be corrected to **540**.

*   **Item: EF-3, Field: 배기풍량(CMH)**
    *   The value entered in the model is 12.
    *   The equipment schedule [2] indicates the airflow for `EF-3` is 12 CMM. The model requires the unit to be CMH. Based on the unit conversion standard [1], 1 CMM = 60 CMH.
    *   Therefore, the `배기풍량(CMH)` must be corrected to **720**.

*   **Item: EF-1, EF-2, EF-3, Field: 배기정압(Pa)**
    *   The `배기정압(Pa)` field was left empty.
    *   The equipment schedule [2] does not contain static pressure information for this equipment. According to the modeling manual [1], a default value of 98 Pa must be entered when the information is unavailable.
    *   Therefore, the value **98** must be entered in the `배기정압(Pa)` field for all three devices.

*   **Item: 1WAY, 2WAY, 4WAY**
    *   This equipment is currently modeled in the `공조기기` section.
    *   The equipment schedule [3] identifies this equipment as `천장카세트` (Ceiling Cassette) type EHP indoor units. The modeling manual [1] specifies that EHP indoor units must be modeled in the `기계설비부문_실내단말기` section, not the `공조기기` section.
    *   Therefore, all entries for `1WAY`, `2WAY`, and `4WAY` must be **deleted** from the `기계설비부문_공조기기` section.
