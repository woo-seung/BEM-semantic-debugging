# Review Report

### 기계설비부문_조명기기 (Mechanical Equipment Section_Lighting Equipment)

*   **Item: 하부 개방 매입등 (무용접슬림형)**
    *   **Field: `조명기기종류` (Lighting Fixture Type)**
        *   The input value is `형광등`, but according to the lighting fixture schedule, all fixtures are `LED` [2, 3].
        *   **Correction:** Please change the `조명기기종류` from `형광등` to `LED`.
    *   **Field: `조명전력(W)` (Lighting Power (W))**
        *   The input value is 10 W. However, the lighting fixture schedule specifies "2/32W", which corresponds to a power of 64 W [2, 3].
