# Review Report

## 건축부문 외피_형별성능내역 (Architectural Sector Envelope_Performance by Type)

### Envelope Type: W1
*   **Field:** `열교방지구조` (Thermal Bridge Prevention Structure)
    *   **User Input:** `false`
    *   **Correction:** The value should be changed to `true`.
    *   **Reason:** The envelope performance drawing shows that the insulation (`비드법보온판`) is installed on the exterior side of the structural frame (`철근콘크리트`). According to the review guidelines, the thermal bridge prevention structure option should be selected in this case [1].

### Envelope Type: F2
*   **Field:** `바닥난방여부` (Floor Heating Status)
    *   **User Input:** `false`
    *   **Correction:** The value should be changed to `true`.
    *   **Reason:** The heating piping plan shows that underfloor heating is installed in the rooms where this floor type is applied [2, 3].
*   **Field:** `열교방지구조` (Thermal Bridge Prevention Structure)
    *   **User Input:** `false`
    *   **Correction:** The value should be changed to `true`.
    *   **Reason:** The envelope performance drawing shows that the insulation (`비드법보온판`) is installed on the exterior side (below the `콘크리트 슬라브`). According to the review guidelines, the thermal bridge prevention structure option should be selected for an envelope in direct contact with the outside air when insulation is on the exterior side of the structural frame [1].

## 건축부문 외피_형별성능내역_재료 (Architectural Sector Envelope_Performance by Type_Materials)

### Envelope Type: W1
*   **Material:** `비드법보온판1종1호`
    *   **Field:** `두께(mm)` (Thickness)
    *   **User Input:** `1500`
    *   **Correction:** The value should be changed to `150`.
    *   **Reason:** The envelope performance drawing specifies the thickness of this material as 0.150 m, which is equivalent to 150 mm.

### Envelope Type: F1
*   **Material:** `콘크리트(1:2:4)` (Composition Order 4)
    *   **Field:** `열전도율(W/mK)` (Thermal Conductivity)
    *   **User Input:** `1.4` (with `열전도율 커스텀 입력 여부` set to `N`)
    *   **Correction:** The thermal conductivity value should be corrected to the material's default value of `1.6`.
    *   **Reason:** The `열전도율 커스텀 입력 여부` (Custom Thermal Conductivity Input Status) is set to `N`, which means the program's default value should be used. The default thermal conductivity for the material `콘크리트(1:2:4)` is 1.6 W/mK [1]. The currently entered value of 1.4 W/mK is incorrect.

## 건축부문 외피 (Architectural Sector Envelope)

### Envelope Type: W2
*   **Field:** `방위` (Orientation)
    *   **User Input:** `일사없음` (No Solar Radiation) for all W2 walls.
    *   **Correction:** The orientation for the W2 walls must be updated based on the insulation area calculation table.
    *   **Reason:** The W2 wall type is defined as being in direct contact with the outside air (`외벽(직접)`). Therefore, it must have a specific orientation (e.g., North, South, East, West) rather than `일사없음` [1]. The correct orientations are found in the insulation area calculation table [4].
        *   The W2 wall with an area of `38.61` m² should have its orientation changed to `남`.
        *   The W2 walls with areas of `3.24` m² and `9.72` m² should have their orientation changed to `동`.
        *   The W2 walls with areas of `4.32` m² and `19.44` m² should have their orientation changed to `서`.

### Envelope Item: 바닥(최하층)
*   **Field:** `면적(m²)` (Area)
    *   **User Input:** `224.91`
    *   **Correction:** The value should be changed to `216`.
    *   **Reason:** The insulation area calculation table for the 1st floor shows the area for the F1 floor type is 216 m² [4].

### Envelope Item: 바닥(피트하부)
*   **Field:** `면적(m²)` (Area)
    *   **User Input:** `26.73`
    *   **Correction:** The value should be changed to `35.64`.
    *   **Reason:** The insulation area calculation table for the 1st floor shows the area for the F2 floor type is 35.64 m² [4].
