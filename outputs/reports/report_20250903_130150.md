# Review Report

### Window G1

*   **Field:** `일사에너지투과율` (Solar Heat Gain Coefficient)
*   **Incorrect Input:** 0.718
*   **Reason for Correction:** The architectural drawing specifies that window G1 is double glazing with an 8mm air gap ('복층유리, 8mm 공기층') [1]. Since a test certificate was not provided, the SHGC must be determined according to the standard table [2]. This table does not list an 8mm air gap, so the value must be calculated by linear interpolation between the values for a 6mm air gap (SHGC 0.717) and a 12mm air gap (SHGC 0.719). The input value of 0.718 is incorrect.
*   **Correction:** The `일사에너지투과율` must be changed to **0.7177**.

### Windows G2 and G3

*   **Field:** `일사에너지투과율` (Solar Heat Gain Coefficient)
*   **Incorrect Input:** 0.471
*   **Reason for Correction:** The architectural drawing specifies that windows G2 and G3 are quadruple glazing with a 12mm air gap and Low-E glass ('사중창, 12mm 공기층, 로이유리') [1]. According to the standard table, the correct SHGC for this configuration is 0.474 [2]. The input value of 0.471 corresponds to a 16mm air gap, which contradicts the drawing.
*   **Correction:** For both G2 and G3, the `일사에너지투과율` must be changed to **0.474**.
