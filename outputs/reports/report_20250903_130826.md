# Review Report

### 기계설비부문_난방기기 (Mechanical Equipment_Heating Equipment)

*   **Item: EHPS - 14**
    *   **Field: 성적계수(COP)**
        *   The user has entered a value of `2`.
        *   However, the `성적계수(COP)` must be calculated based on the equipment's performance data. According to the equipment schedule, the `정격난방용량` is 43.01 kW and the `소비전력` is 13.0 kW [1].
        *   Based on the calculation guideline (정격난방용량 / 소비전력) [4], the correct value is 3.31 (43.01 kW / 13.0 kW).
        *   **Correction**: Please change the `성적계수(COP)` from `2` to `3.31`.

*   **Item: B-1**
    *   **Field: 기기효율(%)**
        *   The user has entered a value of `70`.
        *   However, the `기기효율(%)` must be calculated based on the equipment's performance data. The equipment schedule shows a `정격난방용량` of 16.86 kW, a `연료소비량` of 1.97 L/h, and a `연료발열량` of 8300 kcal/L [2].
        *   Based on the calculation guideline (정격난방용량 / (연료소비량 x 연료발열량)) [4], the correct value is 88.66% (16.86 kW / (1.97 L/h * 9.65 kWh/L)).
        *   **Correction**: Please change the `기기효율(%)` from `70` to `88.66`.

*   **Item: YH-1**
    *   **Field: Equipment Entry**
        *   The user has included the `YH-1` electric radiator in the model.
        *   However, the equipment schedule states that the purpose of this device is `화장실 동파방지` (frost prevention for the toilet) [3].
        *   According to the review manual, equipment used for frost prevention is not subject to evaluation and should not be included in the model [4].
        *   **Correction**: Please delete the entire entry for `YH-1` from the heating equipment list.

        *   The user has entered two identical items for `EHPS - 14`.
        *   The equipment schedule confirms that there is only one unit (`대수`: 1) of this equipment [1].
        *   **Correction**: Please delete one of the two duplicate entries for `EHPS - 14`.

*   **Item: EHPS - 14 (Remaining Entry)**
    *   **Field: 열성능비(COP)**
        *   The user has entered a value of `2`.
        *   However, the `열성능비(COP)` must be calculated based on the equipment's performance data. According to the equipment schedule, the `정격냉방용량` is 34,916 kcal/Hr (40.61 kW) and the `소비전력` during cooling is 13.5 kW [1].
        *   Based on the calculation guideline (정격냉방용량 / 소비전력) [5], the correct value is 3.01 (40.61 kW / 13.5 kW).
        *   **Correction**: For the remaining `EHPS - 14` entry, please change the `열성능비(COP)` from `2` to `3.01`.
