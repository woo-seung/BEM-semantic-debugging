# Review Report

### 외피_형별성능내역 (Envelope Performance Types)

*   **Item: F2, Field: 바닥난방여부 (Floor Heating Status)**
    *   The model input for `바닥난방여부` is `false`.
    *   However, the 2nd floor heating piping plan indicates that a radiant floor heating system is installed in the area corresponding to F2 [2].
    *   The `바닥난방여부` for F2 must be changed to `true`.

*   **Item: W1, Field: 열교방지구조 (Thermal Bridge Prevention Structure)**
    *   The model input for `열교방지구조` is `false`.
    *   According to the cross-section drawing, the insulation for W1 is located on the exterior side of the main structural element (reinforced concrete), which qualifies as a thermal bridge prevention structure [3].
    *   The `열교방지구조` for W1 must be changed to `true`.

*   **Item: W2, Field: 열교방지구조 (Thermal Bridge Prevention Structure)**
    *   The model input for `열교방지구조` is `false`.
    *   According to the cross-section drawing, the insulation for W2 is located on the exterior side of the main structural element (reinforced concrete), which qualifies as a thermal bridge prevention structure [3].
    *   The `열교방지구조` for W2 must be changed to `true`.

*   **Item: F2, Field: 열교방지구조 (Thermal Bridge Prevention Structure)**
    *   The model input for `열교방지구조` is `false`.
    *   According to the cross-section drawing, the insulation for F2 is located on the exterior side of the main structural element (concrete slab), which qualifies as a thermal bridge prevention structure [3].
    *   The `열교방지구조` for F2 must be changed to `true`.

### 외피_형별성능내역_재료 (Material Composition)

*   **Item: W1, Material: 비드법보온판1종1호, Field: 두께(mm) (Thickness)**
    *   The model input for the thickness of `비드법보온판1종1호` is `1500` mm.
    *   The design drawing specifies the thickness of this material as `150` mm [1].
    *   The thickness for `비드법보온판1종1호` in the W1 composition must be corrected to `150`.

### 외피 (Envelope)

*   **Item: W2, Field: 방위 (Orientation)**
    *   The model input for the `방위` of all W2 wall sections is `일사없음`.
    *   W2 is an exterior wall in direct contact with the outside air and therefore must be assigned a cardinal orientation (e.g., South, East, West) to account for solar radiation [1].
    *   The `방위` for the W2 wall sections must be corrected as follows based on the area calculation tables [4, 5]:
        *   The entry with an area of `38.61` m² must be changed to `남`.
        *   The entries with areas of `3.24` m² and `9.72` m² must be changed to `동`.
        *   The entries with areas of `4.32` m² and `19.44` m² must be changed to `서`.

*   **Item: F1, Field: 면적(m²) (Area)**
    *   The model input for the area of `바닥(최하층)` (F1) is `224.91` m².
    *   The area calculation table specifies the area for F1 as `216.00` m² [4].
    *   The area for F1 must be corrected to `216.00`.

*   **Item: F2, Field: 면적(m²) (Area)**
    *   The model input for the area of `바닥(피트하부)` (F2) is `26.73` m².
    *   The area calculation table specifies the area for F2 as `35.64` m² [5].
    *   The area for F2 must be corrected to `35.64`.

*   **Item: R1, Field: 면적(m²) (Area)**
    *   The model has two entries for R1 (`지붕(1층)` and `지붕(2층)`) with a total area of `251.64` m² (177.06 m² + 74.58 m²).
    *   The area calculation tables specify a total roof area of `223.92` m² [4, 5].
    *   The areas for R1 must be corrected so that their sum equals `223.92` m².
