# Review Report

## 창 (Windows)

### 외피_형별성능내역 (Envelope Performance Details)

*   **Item:** G1
    *   **Field:** `일사에너지투과율` (Solar Heat Gain Coefficient)
    *   **Issue:** The input value is incorrect. As no test certificate was provided, the value must follow the standard for the window's composition as defined in the '건축물의 에너지절약설계기준' [1].
    *   **Model Input:** 0.718
    *   **Required Value:** 0.719
    *   **Correction:** Please change the `일사에너지투과율` for G1 from 0.718 to 0.719.

*   **Item:** G2, G3
    *   **Field:** `일사에너지투과율` (Solar Heat Gain Coefficient)
    *   **Issue:** The input value is incorrect. As no test certificate was provided, the value must follow the standard for the window's composition as defined in the '건축물의 에너지절약설계기준' [1]. The drawing [2] specifies a double-glazed unit with a 12mm air gap and soft-coating Low-E glass.
    *   **Model Input:** 0.471
    *   **Required Value:** 0.581
    *   **Correction:** Please change the `일사에너지투과율` for G2 and G3 from 0.471 to 0.581.

### 외피_형별성능내역_재료 (Envelope Performance Details - Materials)

*   **Item:** G1
    *   **Issue:** The material composition for this window is missing. The materials that make up the window must be entered according to the drawing [2].
    *   **Correction:** Please add the following three material layers for window G1:
        1.  `재료명`: 칼라유리, `두께`: 4mm
        2.  `재료명`: 공기층, `두께`: 8mm
        3.  `재료명`: 칼라유리, `두께`: 4mm

*   **Item:** G2, G3
    *   **Issue:** The material composition for these windows is missing. The drawing [2] specifies a quadruple-glazed window (`사중창`) composed of two double-glazed units. This composition must be entered in the model.
    *   **Correction:** Please add the material layers for windows G2 and G3 to represent two sets of the following composition:
        1.  `재료명`: 일반유리, `두께`: 5mm
        2.  `재료명`: 공기층, `두께`: 12mm
        3.  `재료명`: 로이유리, `두께`: 5mm
