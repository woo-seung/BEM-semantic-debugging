# Review Report

### 기계설비부문_공조기기 (Mechanical Equipment_Air Handling Units)

*   **Issue**: Certain items are incorrectly classified as `공조기기` (Air Handling Units).
*   **Description**: The items listed in the model as `1WAY`, `2WAY`, and `4WAY` are identified as `천장카세트` (Ceiling Cassette) units in the design drawings [3]. These are indoor terminal units and should not be entered in the `공조기기` section.
*   **Correction**: You must delete the `1WAY`, `2WAY`, and `4WAY` units from the `기계설비부문_공조기기` list. These items should be moved to the `기계설비부문_실내단말기` section as described below.

### 기계설비부문_실내단말기 (Mechanical Equipment_Indoor Terminal Units)

*   **Issue**: The `실내단말기` (Indoor Terminal Units) section is empty, and the fan power for these units was incorrectly entered in another section.
*   **Description**: The model does not contain any entries for indoor terminal units. The ceiling cassette units (`1WAY`, `2WAY`, `4WAY`) identified in the drawings must be added here. The `팬동력` (Fan Power) for these units should be based on the `소비입력` (Power Consumption) value from the equipment list, as per the manual [1, 3]. The values previously entered in the `공조기기` section were incorrect.
*   **Correction**: Add the following indoor terminal units to the `기계설비부문_실내단말기` section with the correct `팬동력` values.

| Item Name | Field | Model Input | Correct Value |
| :--- | :--- | :--- | :--- |
| 1WAY (for EHP-0.8) | 팬동력(W) | 2000 | 35 |
| 1WAY (for EHP-1.2) | 팬동력(W) | 3200 | 40 |
| 2WAY | 팬동력(W) | 5200 | 70 |
| 4WAY | 팬동력(W) | 7200 | 97 |

*Note: The model input values are taken from the incorrect `송풍기 동력` entries in the `공조기기` section and are shown here for comparison. You must create new entries in the `실내단말기` section with the correct values.*
