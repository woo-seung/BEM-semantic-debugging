### 기계설비부문_난방기기 (Mechanical Equipment_Heating Equipment)

*   **Item: EHPS - 14**
    *   **Field: 성적계수(COP)**
        *   The input value is **2**, but this is incorrect. Based on the equipment's rated heating capacity of 43.01 kW (36,980 kcal/hr) and power consumption of 13.0 kW, the performance coefficient must be calculated as "Rated Heating Capacity / Power Consumption" [1, 2].
        *   Please correct the value to **3.31**.

*   **Item: B-1**
    *   **Field: 기기효율(%)**
        *   The input value is **70**, but this is incorrect. Based on the equipment's rated heating capacity of 14,500 kcal/h, fuel consumption of 1.97 L/h, and fuel calorific value of 8300 kcal/L, the efficiency must be calculated as "(Rated Heating Capacity / (Fuel Consumption × Fuel Calorific Value)) × 100" [1, 2].
        *   Please correct the value to **88.68**.

*   **Item: YH-1**
    *   This equipment, an `전기라디에이터` (electric radiator), is listed for the purpose of `화장실 동파방지` (frost prevention in the restroom) in the design documents [2]. According to the review guidelines, equipment used for frost prevention is not an evaluation target and should not be included in the model [1].
    *   Please **delete the entire entry for YH-1**.
