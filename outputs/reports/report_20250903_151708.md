# Review Report

## 기계설비부문_공조기기 (Mechanical Equipment_Air Handling Units)

### Item: 1WAY, 2WAY, 4WAY
*   **Field:** Equipment Classification
*   **Issue:** These devices are incorrectly classified as `공조기기`. The equipment schedule identifies them as EHP indoor units ('천장카세트') [2]. According to the manual, devices that receive a heat medium to perform heating and cooling, such as EHP indoor units, must be modeled in the `기계설비부문_실내단말기` section [1].
*   **Correction:** Delete the '1WAY', '2WAY', and '4WAY' devices from the `기계설비부문_공조기기` section. They must be added to the `기계설비부문_실내단말기` section as detailed in the next part of this report.

### Item: EF-1
*   **Field:** `배기풍량(CMH)` (Exhaust Airflow)
*   **Issue:** The entered value is 3.5. The equipment schedule specifies the airflow as 3.5 CMM, which is equivalent to 210 CMH [3]. The model requires the value in CMH.
*   **Correction:** Change the `배기풍량(CMH)` from 3.5 to 210.
*   **Field:** `배기정압(Pa)` (Exhaust Fan Static Pressure)
*   **Issue:** The value is not entered. The equipment schedule does not provide this information [3]. The manual states that if the static pressure information is unavailable, a value of 98 Pa should be entered [1].
*   **Correction:** Enter 98 for the `배기정압(Pa)`.
