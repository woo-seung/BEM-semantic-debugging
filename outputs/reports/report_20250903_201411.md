# Review Report

## 기계설비부문_공조기기 (Air Handling Units)

### 1. Equipment Miscategorization: 1WAY, 2WAY, 4WAY Units

The equipment items described as `1WAY`, `2WAY`, and `4WAY` have been incorrectly modeled in the `기계설비부문_공조기기` section. According to the mechanical equipment list, these are `실내기` (Indoor units) and should only be modeled in the `기계설비부문_실내단말기` section [2].

- **Correction**: Delete all equipment items described as `1WAY`, `2WAY`, and `4WAY` from the `기계설비부문_공조기기` section.

### 2. Exhaust Fan: EF-1

- **배기풍량 (Exhaust Airflow Rate)**: The input value is incorrect due to a unit mismatch. The equipment list specifies the airflow rate in CMM (Cubic Meters per Minute), but the model requires CMH (Cubic Meters per Hour).
    - **Model Input**: 3.5 CMH
    - **Evidence**: 3.5 CMM [1]
    - **Correction**: The value must be converted from CMM to CMH (3.5 CMM * 60 min/hr). Change the `배기풍량` to **210** CMH.

- **배기정압 (Exhaust Static Pressure)**: The static pressure value is missing. According to the review guidelines, if this information is not available in the design documents, a default value should be used [5].
    - **Model Input**: Not entered
    - **Evidence**: Not specified in the equipment list [1]
    - **Correction**: Enter **98** Pa for the `배기정압`.

### 3. Exhaust Fan: EF-2

- **대수 (Quantity)**: The number of installed units is incorrect.
    - **Model Input**: 1
    - **Evidence**: 2 [3]
    - **Correction**: Change the `대수` to **2**.

- **배기풍량 (Exhaust Airflow Rate)**: The input value is incorrect due to a unit mismatch. The equipment list specifies the airflow rate in CMM, but the model requires CMH.
    - **Model Input**: 9 CMH
    - **Evidence**: 9 CMM [1]
    - **Correction**: The value must be converted from CMM to CMH (9 CMM * 60 min/hr). Change the `배기풍량` to **540** CMH.

- **배기정압 (Exhaust Static Pressure)**: The static pressure value is missing. According to the review guidelines, if this information is not available in the design documents, a default value should be used [5].
    - **Model Input**: Not entered
    - **Evidence**: Not specified in the equipment list [1]
    - **Correction**: Enter **98** Pa for the `배기정압`.

### 4. Exhaust Fan: EF-3

- **대수 (Quantity)**: This equipment is modeled but does not exist in the building design.
    - **Model Input**: 1
    - **Evidence**: 0 units are installed according to the mechanical drawings [3].
    - **Correction**: Delete the `EF-3` equipment item from the model.
