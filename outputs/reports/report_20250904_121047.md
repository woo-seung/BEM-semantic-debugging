# Review Report

### 기계설비부문_공조기기 (Mechanical Equipment_Air Handling Units)

*   **Items: 1WAY, 2WAY, 4WAY**
    *   **Issue:** These items are incorrectly modeled in the '공조기기' section. The equipment schedule identifies them as EHP indoor units ('천장카세트') [3]. According to the manual, devices that receive heat medium to perform heating and cooling via heat exchange with indoor air must be modeled as '실내단말기' (Indoor Terminal Units) [1].
    *   **Correction:** Delete all items described as '1WAY', '2WAY', and '4WAY' from the '기계설비부문_공조기기' section. These items must be added to the '기계설비부문_실내단말기' section as detailed below.

*   **Item: EF-1**
    *   **Field: 배기풍량 (Exhaust Airflow)**
        *   **User Input:** `3.5`
        *   **Evidence:** The equipment schedule specifies an airflow of `3.5 CMM` [2].
        *   **Correction:** The model requires the airflow unit to be CMH. Based on the conversion factor `1 CMM = 60 CMH`, the value must be corrected to `210` [1].
    *   **Field: 배기정압 (Exhaust Static Pressure)**
        *   **User Input:** (empty)
        *   **Evidence:** The static pressure is not specified in the equipment schedule [2].
        *   **Correction:** According to the manual, if the static pressure information is not available, `98 Pa` must be entered [1].

*   **Item: EF-2**
    *   **Field: 배기풍량 (Exhaust Airflow)**
        *   **User Input:** `9`
        *   **Evidence:** The equipment schedule specifies an airflow of `9 CMM` [2].
        *   **Correction:** The model requires the airflow unit to be CMH. Based on the conversion factor `1 CMM = 60 CMH`, the value must be corrected to `540` [1].
    *   **Field: 배기정압 (Exhaust Static Pressure)**
        *   **User Input:** (empty)
        *   **Evidence:** The static pressure is not specified in the equipment schedule [2].
        *   **Correction:** According to the manual, if the static pressure information is not available, `98 Pa` must be entered [1].

*   **Item: EF-3**
    *   **Field: 배기풍량 (Exhaust Airflow)**
        *   **User Input:** `12`
        *   **Evidence:** The equipment schedule specifies an airflow of `12 CMM` [2].
        *   **Correction:** The model requires the airflow unit to be CMH. Based on the conversion factor `1 CMM = 60 CMH`, the value must be corrected to `720` [1].
    *   **Field: 배기정압 (Exhaust Static Pressure)**
