# Review Report

## 건축부문 외피 (Building Envelope)

*   **Item**: All `벽체` items with `형별성능내역` set to `W2`
    *   **Field**: 방위 (Orientation)
    *   **Description**: The `방위` for wall type `W2` is incorrectly entered as `일사없음`. The component `W2` is defined as a `외벽(직접)` (directly exposed exterior wall), which must receive solar radiation. The `일사없음` setting is only for walls that are not exposed to the outside.
        *   **Model Input**: 일사없음
        *   **Correction**: The `방위` for all `W2` walls must be changed from `일사없음` to the correct cardinal direction (e.g., 동, 서, 남, 북) as indicated on the architectural drawings.

## 건축부문 외피_형별성능내역 (Building Envelope_Type Performance Details)

*   **Item**: W1
    *   **Field**: 열교방지구조 (Thermal Bridge Prevention)
    *   **Description**: The `열교방지구조` is incorrectly set to `false`. The architectural drawings show that the insulation (`비드법보온판`) is installed on the exterior side of the structural frame (`철근콘크리트`), which qualifies as a thermal bridge prevention structure.
        *   **Model Input**: false
        *   **Correction**: The `열교방지구조` checkbox for `W1` must be checked (`true`).

*   **Item**: F1
    *   **Field**: 바닥난방여부 (Floor Heating)
    *   **Description**: The `바닥난방여부` is incorrectly set to `false`. The heating piping plan confirms that radiant floor heating is installed in this area.
        *   **Model Input**: false
        *   **Correction**: The `바닥난방여부` checkbox for `F1` must be checked (`true`).

*   **Item**: F2
    *   **Field**: 바닥난방여부 (Floor Heating)
    *   **Description**: The `바닥난방여부` is incorrectly set to `false`. The heating piping plan confirms that radiant floor heating is installed in this area.
        *   **Model Input**: false
        *   **Correction**: The `바닥난방여부` checkbox for `F2` must be checked (`true`).

## 건축부문 외피_형별성능내역_재료 (Building Envelope_Type Performance Details_Materials)

*   **Item**: W1
    *   **Field**: 두께(mm) (Thickness)
    *   **Description**: The thickness of the insulation material `비드법보온판1종1호` is incorrect.
        *   **Model Input**: 1500 mm
        *   **Drawing Value**: 150 mm
        *   **Correction**: The `두께` for the material `비드법보온판1종1호` must be changed from `1500` to `150`.
