# Review Report

### 기계설비부문_공조기기 (Air Handling Units)

*   **Item: EF-1**
    *   **Field: 배기풍량 (Exhaust Airflow)**
        *   The input value is `3.5` CMH.
        *   According to the equipment list, the correct airflow is `210` CMH [2].
        *   **Correction:** The value must be changed from `3.5` to `210`. The input unit for airflow is cubic meters per hour (CMH), but the value was entered in cubic meters per minute (CMM).

*   **Item: EF-2**
    *   **Field: 배기풍량 (Exhaust Airflow)**
        *   The input value is `9` CMH.
        *   According to the equipment list, the correct airflow is `540` CMH [2].
        *   **Correction:** The value must be changed from `9` to `540`. The input unit for airflow is cubic meters per hour (CMH), but the value was entered in cubic meters per minute (CMM).

*   **Item: EF-3**
    *   **Field: 배기풍량 (Exhaust Airflow)**
        *   The input value is `12` CMH.
        *   According to the equipment list, the correct airflow is `720` CMH [2].
        *   **Correction:** The value must be changed from `12` to `720`. The input unit for airflow is cubic meters per hour (CMH), but the value was entered in cubic meters per minute (CMM).

*   **Items: 1WAY, 2WAY, 4WAY**
    *   **Field: 공조방식 (Air Conditioning System)**
        *   These items are entered as `변풍량` (Variable Air Volume) air handling units.
        *   The equipment list identifies these as EHP indoor units (천장카세트) [3]. According to the manual, devices that use a heat medium to provide heating and cooling, such as EHP indoor units, must be modeled in the `기계설비부문_실내단말기` (Indoor Terminal Units) section, not as `공조기기` (Air Handling Units) [1].
        *   **Correction:** All entries for `1WAY`, `2WAY`, and `4WAY` must be deleted from the `기계설비부문_공조기기` section. They must be added to the `기계설비부문_실내단말기` section as detailed below.
