# Review Report

### Walls (벽체)

*   **Item:** `외피` (Envelope)
*   **Field:** `방위` (Orientation)
*   **Description:** The orientation for wall entries using the `W2` performance type is incorrectly set to `일사없음` (No Solar Radiation). According to the `외피전개도`, these walls are directly exposed to the outside and must have specific orientations assigned [1]. The `방위` for the following entries must be corrected:
    *   For the entry with `형별성능내역` `W2` and `면적` 38.61 m², the `방위` was entered as `일사없음`. It must be corrected to `남`.
    *   For the entry with `형별성능내역` `W2` and `면적` 3.24 m², the `방위` was entered as `일사없음`. It must be corrected to `동`.
    *   For the entry with `형별성능내역` `W2` and `면적` 9.72 m², the `방위` was entered as `일사없음`. It must be corrected to `동`.
    *   For the entry with `형별성능내역` `W2` and `면적` 4.32 m², the `방위` was entered as `일사없음`. It must be corrected to `서`.
    *   For the entry with `형별성능내역` `W2` and `면적` 19.44 m², the `방위` was entered as `일사없음`. It must be corrected to `서`.

### Floors (바닥)

*   **Item:** `외피` (Envelope)
*   **Field:** `면적` (Area)
*   **Description:** The area for the floor entry using the `F1` performance type is incorrect.
    *   **User Input:** 224.91 m²
    *   **Evidence:** According to the `외피전개도`, the correct area is 216 m² [1].
    *   **Correction:** The `면적` for the entry with `형별성능내역` `F1` must be changed to 216 m².

*   **Item:** `외피` (Envelope)
*   **Field:** `면적` (Area)
*   **Description:** The area for the floor entry using the `F2` performance type is incorrect.
    *   **User Input:** 26.73 m²
    *   **Evidence:** According to the `외피전개도`, the correct area is 35.64 m² [1].
    *   **Correction:** The `면적` for the entry with `형별성능내역` `F2` must be changed to 35.64 m².
