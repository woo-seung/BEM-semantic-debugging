# Review Report

### 건축부문 외피_형별성능내역 (Building Part Envelope_Type Performance Details)

*   **Item**: Window Types `G2`, `G3`
*   **Field**: `일사에너지투과율` (Solar Heat Gain Coefficient, SHGC)
*   **Issue**: The `일사에너지투과율` values for window types `G2` and `G3` are incorrect. Since the drawings do not provide a specific SHGC value, the default value from the '건축물의 에너지절약설계기준' must be used [1]. The composition for these windows is '사중창' (quadruple glazing) with '로이유리' (Low-E glass) and a 12mm air gap [2]. The standard specifies an SHGC of 0.474 for this configuration [1].
    *   **User Input**: 0.471
    *   **Correct Value**: 0.474
*   **Correction**: In the `건축부문 외피_형별성능내역` table, change the `일사에너지투과율` for window types `G2` and `G3` from 0.471 to **0.474**.

### 건축부문 외피 (Building Part Envelope)

*   **Item**: Missing Door on 1st Floor
*   **Field**: `면적(m²)` (Area)
*   **Issue**: The model is missing a door on the south-facing side of the 1st floor. The area calculation sheet for the `보건지소` shows a door designated as `D3` with an area of 8.64 m² on the south facade, which has not been included in the model [3].
    *   **User Input**: The total area for south-facing doors on the 1st floor is 2.1 m².
    *   **Correct Value**: The total area for south-facing doors on the 1st floor should be 10.74 m².
*   **Correction**: Add a new door to the `건축부문 외피` table with the following properties:
    *   **형별성능내역**: D3
    *   **방위**: 남 (South)
    *   **면적(m²)**: 8.64
