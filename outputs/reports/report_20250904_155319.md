# Review Report

### 기계설비부문_공조기기 (Mechanical Equipment_Air Handling Units)

*   **Misclassification of Indoor Units**
    *   **Items:** 1WAY, 2WAY, 4WAY
    *   **Error:** These devices are incorrectly modeled in the `공조기기` section. The equipment schedule [2] identifies them as Fan Coil Units (FCU-1, FCU-2, FCU-3, FCU-4). According to modeling guidelines, devices that use a heat medium to condition room air, such as FCUs, must be modeled in the `기계설비부문_실내단말기` (Indoor Terminal Units) section [1].
    *   **Correction:** Delete the four entries corresponding to '1WAY', '2WAY', and '4WAY' from the `공조기기` section. These devices must be added to the `기계설비부문_실내단말기` section, as detailed in the next part of this report.

*   **Incorrect Airflow Rate for Ventilation Fan EF-1**
    *   **Item:** EF-1, **Field:** `배기풍량`
    *   **Input Value:** 3.5 CMH
    *   **Evidence:** The equipment schedule lists the airflow rate as 3.5 CMM [2].
    *   **Error:** The model requires the airflow rate unit to be CMH (Cubic Meters per Hour), but the value was entered using the CMM (Cubic Meters per Minute) value from the schedule without conversion [1].
    *   **Correction:** The value must be converted using the formula 1 CMM = 60 CMH. Please change the `배기풍량` to **210** CMH.

*   **Incorrect Airflow Rate for Ventilation Fan EF-2**
    *   **Item:** EF-2, **Field:** `배기풍량`
    *   **Input Value:** 9 CMH
    *   **Evidence:** The equipment schedule lists the airflow rate as 9 CMM [2].
    *   **Error:** The model requires the airflow rate unit to be CMH, but the value was entered using the CMM value from the schedule without conversion [1].
    *   **Correction:** Please change the `배기풍량` to **540** CMH.

*   **Incorrect Airflow Rate for Ventilation Fan EF-3**
    *   **Item:** EF-3, **Field:** `배기풍량`
    *   **Input Value:** 12 CMH
    *   **Evidence:** The equipment schedule lists the airflow rate as 12 CMM [2].
    *   **Error:** The model requires the airflow rate unit to be CMH, but the value was entered using the CMM value from the schedule without conversion [1].
    *   **Correction:** Please change the `배기풍량` to **720** CMH.
