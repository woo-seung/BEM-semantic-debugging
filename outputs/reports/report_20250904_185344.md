# Review Report

## 기계설비부문_난방기기

### **Item: YH-1**
*   **Issue:** The equipment 'YH-1' is included in the evaluation, but its purpose is for anti-freeze, which is not an evaluation target.
*   **Details:**
    *   The purpose of 'YH-1' is listed as '화장실 동파방지' (anti-freeze for toilets) in the equipment schedule [2].
    *   According to the evaluation guidelines, equipment used for anti-freeze purposes should be excluded from the model [1].
*   **Correction:**
    *   The equipment 'YH-1' must be removed from the '기계설비부문_난방기기' list.

### **Item: EHPS - 14**
*   **Field:** `성적계수(COP)`
*   **Issue:** The input value for the Coefficient of Performance (COP) is incorrect.
*   **Details:**
    *   **Input Value:** 2
    *   **Evidence-Based Value:** The COP is calculated as 'Rated Heating Capacity (kW) / Power Consumption (kW)' [1]. Based on the equipment schedule, the capacity is 43.0 kW (36,980 kcal/Hr) and the power consumption is 13.0 kW [2]. The calculated COP is 3.31 (43.0 kW / 13.0 kW).
*   **Correction:**
    *   The value for `성적계수(COP)` must be changed from 2 to **3.31**.

### **Item: B-1**
*   **Field:** `기기효율(%)`
*   **Issue:** The input value for the equipment efficiency is incorrect.
*   **Details:**
    *   **Input Value:** 70
    *   **Evidence-Based Value:** The efficiency is calculated as 'Rated Heating Capacity / (Fuel Consumption × Fuel Calorific Value)' [1]. Based on the equipment schedule, the capacity is 14,500 kcal/h, fuel consumption is 1.97 L/h, and the calorific value of the fuel is 8,300 kcal/L [2]. The calculated efficiency is 88.68% ((14,500 kcal/h) / (1.97 L/h × 8,300 kcal/L) × 100).
*   **Correction:**
    *   The value for `기기효율(%)` must be changed from 70 to **88.68**.

*   **Field:** `펌프동력합계(kW)`
*   **Issue:** The input value for the total pump power is incorrect.
*   **Details:**
    *   **Input Value:** 0
    *   **Evidence-Based Value:** The model input is 0 kW, but the '1F Heating Piping Plan' shows that this boiler is connected to one '온수순환펌프' and one '급탕순환펌프' [3]. The total pump power should be the sum of the motor power of these associated pumps, excluding any standby units [1].
*   **Correction:**
    *   The value for `펌프동력합계(kW)` must be corrected from 0 to the sum of the power for the '온수순환펌프' and '급탕순환펌프' as specified in the equipment schedule.

## 기계설비부문_냉방기기

### **Item: EHPS - 14**
*   **Field:** `열성능비(COP)`
*   **Issue:** The input value for the Coefficient of Performance (COP) is incorrect.
*   **Details:**
    *   **Input Value:** 2
    *   **Evidence-Based Value:** The COP is calculated as 'Rated Cooling Capacity (kW) / Power Consumption (kW)' [1]. Based on the equipment schedule, the cooling capacity is 40.6 kW (34,916 kcal/Hr) and the power consumption during cooling is 13.5 kW [2]. The calculated COP is 3.01 (40.6 kW / 13.5 kW).
*   **Correction:**
    *   The value for `열성능비(COP)` must be changed from 2 to **3.01**.
