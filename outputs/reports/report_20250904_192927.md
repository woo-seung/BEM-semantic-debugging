# Review Report

## 기계설비부문_공조기기 (Mechanical Equipment_Air Handling Units)

### 1WAY, 2WAY, 4WAY Units
*   **Issue**: The devices identified as '1WAY', '2WAY', and '4WAY' are EHP indoor units and have been incorrectly modeled in the '공조기기' section. According to the manual, devices that receive a heat medium from a heat source to perform heat exchange with indoor air for heating and cooling, such as EHP indoor units, must be modeled in the '기계설비부문_실내단말기' section [1].
*   **Correction**: Delete all entries for '1WAY', '2WAY', and '4WAY' units from the '공조기기' section. These devices must be added to the '실내단말기' section as detailed in the next chapter of this report.

### EF-1
*   **Field**: `배기풍량(CMH)` (Exhaust Airflow)
*   **Issue**: The input value is incorrect due to a unit mismatch.
*   **Model Input**: 3.5
*   **Evidence**: The equipment schedule lists the airflow as 3.5 CMM [2]. The modeling manual requires this value to be in CMH, using the conversion factor 1 CMM = 60 CMH [1].
*   **Correction**: The value should be 3.5 * 60 = 210. Please correct the `배기풍량(CMH)` to **210**.

*   **Field**: `배기정압(Pa)` (Exhaust Static Pressure)
*   **Issue**: The value is missing.
*   **Model Input**: (empty)
*   **Evidence**: The equipment schedule does not provide a static pressure value [2]. The manual states that if this information is not available, **98 Pa** should be entered [1].
*   **Correction**: Please enter **98** for `배기정압(Pa)`.
