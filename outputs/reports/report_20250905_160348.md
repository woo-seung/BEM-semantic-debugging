# Review Report

### 기계설비부문_공조기기 (Mechanical Equipment_Air Handling Units)

*   **Item: 1WAY, 2WAY, 4WAY**
    *   **Field: General**
        *   **Issue:** The devices described as '1WAY', '2WAY', and '4WAY' are incorrectly modeled in the `기계설비부문_공조기기` section. According to the design drawings, these are '가변형 천정형 냉난방기 - 실내기' (Variable Ceiling-Type Air Conditioner/Heater Indoor Units) [2]. Per the modeling manual, devices that exchange heat to condition indoor air using a heat medium should be modeled as `실내단말기` (Indoor Terminals) [1].
        *   **Correction:** All entries for '1WAY', '2WAY', and '4WAY' devices must be deleted from the `기계설비부문_공조기기` section. They should be added to the `기계설비부문_실내단말기` section as detailed below.

*   **Item: EF-1**
    *   **Field: 배기풍량(CMH) (Exhaust Airflow Rate)**
        *   **Issue:** The model input is 3.5. The design drawings specify an airflow rate of 3.5 CMM, which is equivalent to 210 CMH [2]. The input field requires the value in CMH.
        *   **Correction:** The value must be changed from 3.5 to 210.
    *   **Field: 배기정압(Pa) (Exhaust Static Pressure)**
        *   **Issue:** The model does not have a value for this field. This information is not available in the design drawings [2].
        *   **Correction:** As per the modeling manual, when static pressure information is unavailable, a default value of 98 Pa must be entered [1].

*   **Item: EF-2**
    *   **Field: 배기풍량(CMH) (Exhaust Airflow Rate)**
        *   **Issue:** The model input is 9. The design drawings specify an airflow rate of 9 CMM, which is equivalent to 540 CMH [2]. The input field requires the value in CMH.
        *   **Correction:** The value must be changed from 9 to 540.
    *   **Field: 배기정압(Pa) (Exhaust Static Pressure)**
        *   **Issue:** The model does not have a value for this field. This information is not available in the design drawings [2].
        *   **Correction:** As per the modeling manual, when static pressure information is unavailable, a default value of 98 Pa must be entered [1].

*   **Item: EF-3**
    *   **Field: 배기풍량(CMH) (Exhaust Airflow Rate)**
        *   **Issue:** The model input is 12. The design drawings specify an airflow rate of 12 CMM, which is equivalent to 720 CMH [2]. The input field requires the value in CMH.
        *   **Correction:** The value must be changed from 12 to 720.
    *   **Field: 배기정압(Pa) (Exhaust Static Pressure)**
        *   **Issue:** The model does not have a value for this field. This information is not available in the design drawings [2].
        *   **Correction:** As per the modeling manual, when static pressure information is unavailable, a default value of 98 Pa must be entered [1].

*   **Correction:** The following devices must be added to this section. The `용량(kW)` should be entered based on the rated heating capacity [1].
