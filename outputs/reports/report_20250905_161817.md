# Review Report

## 기계설비부문_조명기기 (Mechanical Equipment Sector_Lighting Fixtures)

### 조명기기 목록 (Lighting Fixture List)
The lighting fixture list entered in the model is incorrect in terms of fixture type (`조명기기종류`), power (`조명전력(W)`), and quantity (`대수`).

-   **Model Input:** The model contains multiple entries for `형광등` (Fluorescent) and `일반등` (General) with a total calculated power of 986W. For example, one entry is for 31 units of 10W `형광등`.
-   **Evidence:** According to the `조명기구일람표` (Lighting Fixture Schedule) on the electrical plans, all specified lighting fixtures are `LED` [1]. The power and quantity values also differ significantly from the model. For instance, the drawings specify 31 units of 64W (2/32W) `매입슬립등` and 50 units of 26W (2/13W) `다운라이트`. The total power calculated from the drawings is 4,016W.
-   **Correction:** All existing lighting fixture entries in the model must be deleted. They should be replaced with the following consolidated list, which is based on the design drawings [1]. The `조명기기종류` for all new entries must be set to `LED`.

| 조명전력(W) | 대수 |
| :---------- | ---: |
| 108         |    2 |
| 72          |    2 |
| 64          |   31 |
| 60          |    3 |
| 26          |   52 |
| 20          |    7 |
